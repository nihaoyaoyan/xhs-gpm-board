# 若溪 · 小红书 GPM 分析

晚风店铺 · 小红书商品笔记绩效日报，基于商家后台导出数据自动生成。

## 📁 目录

```
├── index.html              ← 日报目录首页
├── README.md               ← 本文件
├── reports/
│   └── YYYY-MM-DD/         ← 每日分析（按日期归档）
│       ├── index.html       → 落地页：KPI 摘要 + 导航 + 核心洞察
│       ├── dashboard.html   → 交互看板：图表 + 明细表 + 筛选导出
│       └── report.html      → 分析报告：发现 + 建议 · A4 打印友好
├── data/                    ← 构建脚本与数据（不上传 Git）
├── xhs-gpm/                 ← 旧版参考看板
└── assets/                  ← 公共资源
```
